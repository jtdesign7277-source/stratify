# Signup UI sandbox

Standalone preview of the Stratify signup form (dark theme, OAuth + email/password).

**To view:** Open `index.html` in a browser, or run a local server:

```bash
cd signup-sandbox
python3 -m http.server 5177
```

Then open http://localhost:5177

In Cursor: **Cmd+Shift+P** → "Simple Browser: Show" → enter the URL above.
